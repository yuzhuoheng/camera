# 随拍随存 · MVP 开发计划

## 项目概述
随拍随存是一款微信小程序，定位为“拍照即上云、轻量相册分享”。用户无需在本地整理，照片自动按照时间/相册归档，并可将某个相册分享给微信好友。

## MVP目标
- 一键拍照或选择已有照片并上传云端存储
- 基于时间或手动创建的相册进行归档与浏览
- 将相册生成分享入口，指定好友可查看
- 确保私密数据访问受控，基础费用可控、体验流畅

## 核心用户故事
- 作为用户，我可以打开小程序直接拍照并自动上传到云端
- 作为用户，我可以查看云端中的照片流和相册列表
- 作为用户，我可以创建/重命名相册并将照片归入该相册
- 作为用户，我可以生成相册的分享链接并分享给微信好友
- 作为被分享者，我可以在获得授权后查看相册内容

## MVP范围（功能清单）
- 拍照/选图上传：支持 camera/chooseImage，上传至云存储
- 相册管理：创建/查看相册；相册内照片网格展示，支持删除
- 照片流：按时间倒序展示最近上传
- 分享：相册分享页生成分享令牌，好友打开小程序可查看
- 账户与鉴权：基于 openid；敏感操作校验令牌/权限
- 基础容错与提示：上传失败重试、网络异常提示

## 系统架构
- 前端：微信小程序（WXML/WXSS/JS），使用小程序能力 camera、chooseImage、onShareAppMessage
- 云后端：微信云开发（CloudBase）
  - 云存储：保存图片文件（原图/压缩图，可先只保存原图）
  - 数据库：photos、albums、shares、users 集合
  - 云函数：鉴权校验、分享令牌生成/校验、受控删除

## 数据模型（初版）
- users
  - _id: openid
  - nickname, avatar（可后续补充）
- albums
  - _id
  - ownerOpenid
  - name
  - createdAt
- photos
  - _id
  - ownerOpenid
  - albumId（可为空，表示在“未分组/时间线”）
  - fileId（云存储文件ID）
  - takenAt（拍摄时间）
  - createdAt（上传时间）
- shares
  - _id
  - albumId
  - ownerOpenid
  - token（随机字符串/短码）
  - scope: private（指定openid列表）/ token（持令牌可访问）
  - allowedOpenids（当 scope=private 时使用）
  - createdAt, expiresAt（可选，MVP可不设过期）

## 页面与流程
- 首页（pages/index）
  - 展示相册列表与最近照片流入口
  - 入口按钮：拍照/选图、创建相册
- 拍照/上传页（pages/capture）
  - 使用 camera 或 chooseImage
  - 选择归入相册（或留在时间线）
  - 执行上传，显示进度与结果
- 相册详情页（pages/album）
  - 网格展示该相册照片；支持单张预览、删除（仅owner）
  - 分享入口：生成分享令牌/设置分享范围
- 分享访问页（pages/share）
  - 通过 token 或分享参数进入
  - 云函数校验后拉取可访问的相册内容

## 权限与安全
- 所有照片默认仅 ownerOpenid 可访问
- 分享访问通过云函数校验：匹配 token 或在 allowedOpenids 列表内
- 删除/重命名等敏感操作仅允许 ownerOpenid
- 云数据库/存储安全规则：限制写入/读取到 owner 或被授权访问者

## 分享方案（MVP）
- 使用 onShareAppMessage 在相册详情处生成分享卡片
- 分享参数带上 token，好友进入 pages/share 读取并校验
- MVP先支持“持令牌可访问”的受控分享；后续再支持精确到好友openid白名单

## 云资源与配置
- 开启微信云开发环境，绑定环境ID
- 初始化数据库集合：users、albums、photos、shares
- 配置存储桶与 CDN（默认 CloudBase 存储即可）
- 编写云函数：
  - createShareToken(albumId, scope, allowedOpenids?)
  - validateShareToken(token)
  - secureDeletePhoto(photoId)

## 非功能需求
- 性能：上传进度反馈、列表分页/懒加载
- 可靠性：失败重试、超时提示、弱网兼容
- 隐私：本地不持久化敏感数据；日志不打印文件ID/隐私信息

## 风险与对策
- 存储费用增长：提供基础免费额度与清理策略（如批量归档/压缩）
- 分享被转发导致扩散：令牌长度/复杂度提升，支持关闭分享、撤销令牌
- 端权限绕过：所有可读写操作走云函数校验，数据库规则严格限制

## 验收标准（MVP完成判定）
- 能拍照/选图并成功上传到云端，查看与删除正常
- 能创建相册并在相册内浏览照片
- 能生成分享令牌，好友通过分享入口成功查看相册
- 未授权用户无法访问非公开相册数据
- 异常场景（网络失败/上传错误）有提示与可重试

## 迭代建议（非MVP）
- 自动分类：按时间/地点/人脸聚类
- 智能压缩与原图备份策略
- 批量管理：多选移动、批量删除
- 私密相册加锁（本地PIN或二次校验）
- 分享统计：访问次数、访客列表（满足隐私前提）
