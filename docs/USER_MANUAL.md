## 1. 快速开始

### 登录与授权
首次进入小程序，您会看到登录提示。
*   **立即登录**：建议您登录使用，以便在不同设备间同步您的相册和数据。
*   **暂不登录**：如果您只是受邀查看朋友分享的照片，可以选择暂不登录，直接进入**访客模式**。

## 2. 首页功能

首页是您的操作中心，包含以下板块：

*   **快捷操作栏**：
    *   📷 **拍摄/上传**：直接调用相机拍照或从手机相册选择照片上传。
    *   ➕ **新建相册**：快速创建一个新的分类相册。
    *   🎁 **邀请好友**：分享小程序给好友。
    *   ⚙️ **设置**：个人信息与应用设置。

*   **最近上传**：
    *   展示您最近上传的 5 张照片。
    *   点击照片可进入**高清大图预览**模式，支持左右滑动查看。

*   **最近访问的分享**：
    *   自动记录您访问过的他人分享相册。
    *   点击 **"查看全部"** 可进入历史记录页面，管理或清空您的访问历史。

## 3. 照片与相册管理

### 上传照片
1.  点击首页或相册页底部的 **"拍摄/上传"** 按钮。
2.  选择 **"拍照"** 或 **"从手机相册选择"**。
3.  选择目标相册（默认为当前相册或默认相册），确认上传。

### 管理相册
进入 **"我的相册"** 列表，点击任意相册进入详情：
*   **重命名**：点击右上角菜单，选择 **"修改相册名称"**。
*   **删除相册**：点击右上角菜单，选择 **"删除相册"**（注意：默认相册不可删除，删除相册将同时删除其中的所有照片）。

### 批量操作照片
在相册详情页，您可以进行批量管理：
1.  点击右上角的 **"选择"** 按钮（或长按某张照片）进入选择模式。
2.  点选多张照片。
3.  **下载**：点击底部的下载图标，将选中的原图保存到手机系统相册。
4.  **删除**：点击底部的删除图标，批量移除照片。

## 4. 分享与协作

我们提供安全灵活的分享机制，支持多人共享相册。

### 发起分享
在相册详情页，点击右上角菜单或分享按钮：
1.  **邀请查看**：生成一个**只读**链接，好友只能浏览和下载照片。
2.  **邀请上传**：生成一个**协作**链接，好友不仅可以浏览，还可以向该相册上传照片（适合聚会、婚礼等场景）。

### 管理分享
您可以随时收回分享权限：
1.  在相册详情页点击右上角菜单。
2.  选择 **"分享管理"**。
3.  查看当前生效的分享链接，点击 **"撤销"** 即可立即使该链接失效。

### 访问分享
*   点击好友发送的小程序卡片，即可直接打开对应的相册。
*   系统会自动将该相册加入您的 **"最近访问"** 列表，方便下次快速找到。

---

## 5. 访客模式

如果您没有注册或登录，依然可以使用以下功能：
*   **查看分享**：通过好友分享的链接，您可以完整浏览相册内容。
*   **下载照片**：您可以将照片保存到本地。
*   *注意：新建相册、上传照片、查看历史记录等功能需要登录后才能使用。*

## 6. 常见问题

**Q: 我查看自己分享出去的链接，会被记录在“最近访问”里吗？**
A: 不会。系统会自动识别，只有访问他人分享的相册才会被记录。

**Q: 删除照片后还能恢复吗？**
A: 目前删除操作是永久的，请谨慎操作。

**Q: 为什么我看不到“最近访问”列表？**
A: 只有当您访问过他人的分享链接后，该板块才会显示。
